# AI-Apps: Construyendo con Vibecoding 🚀

Bienvenido a **AI-Apps**, un repositorio donde colecciono aplicaciones experimentales, herramientas y utilidades completas creadas utilizando la filosofía del **Vibecoding**.

## ¿Qué es Vibecoding?

El "Vibecoding" es un enfoque de desarrollo ágil y moderno donde el código y la arquitectura se construyen colaborativamente con agentes de Inteligencia Artificial en lenguaje natural. En lugar de escribir el código línea por línea, el desarrollo se enfoca en dirigir, estructurar instrucciones claras, iterar ideas, y permitir que la IA construya el software directamente.

Esto permite enfocarse en la **experiencia de usuario, el diseño del producto y la lógica de negocio**, acelerando drásticamente el proceso de creación.

## Aplicaciones en este repositorio

A medida que vayamos construyendo nuevas ideas en conjunto, cada nueva aplicación vivirá en su propio directorio dentro de este monorepositorio.

### 1. 🎤 Entrevistador Universal (`/universal-interviewer`)
**[▶️ Probar Aplicación en Vivo](https://alamozuna.github.io/AI-Apps/)**

Una aplicación interactiva en React (con Vite y Tailwind CSS v4) que se integra con **Google Gemini (2.5 Flash)**. Permite subir un Curriculum Vitae en formato PDF junto a la descripción de una vacante, para generar de manera automática 25 preguntas de entrevista altamente personalizadas y estrategias de respuesta para cada una.

### 2. ⚛️ React Example (`/react-example`)
Una aplicación de dashboard de ejemplo en React previa a la organización del monorepositorio.
